# 2urn CV Pro+ Template

**Status:** Live

**Markets:** Professionals, Students, Job Seekers

## Overview

A premium Carrd CV/portfolio template built for professionals who need a fast, credible, mobile-first personal site without the overhead of WordPress or the cost of a Squarespace subscription.

## Features

- Clean typographic hierarchy optimised for CV/résumé content
- Gallery Filter Pro+ integration for portfolio work
- Keyboard Navigation Pro+ for section-by-section browsing
- Clone Engine v2.0 for repeating work/education entries
- Design Tokens system for easy rebrand in under 5 minutes
- Mobile-first, fully responsive
- Carrd Pro+ required ($49/year)

## Sections

1. **Hero** — Name, title, one-line bio
2. **About** — Extended bio, photo
3. **Work** — Experience entries (Clone Engine repeatable)
4. **Education** — Degree entries (Clone Engine repeatable)
5. **Portfolio** — Gallery with Filter Pro+ categories
6. **Skills** — Tag cloud or list layout
7. **Contact** — Form embed + social links

## Plugins Included

- Gallery Filter Pro+ v1.7.3
- Keyboard Navigation Pro+ v2.1.2
- Clone Engine v2.0
- Design Tokens v1.4

## Installation

1. Purchase the template at [2urn.com](https://2urn.com)
2. Duplicate the Carrd template to your account
3. Replace placeholder content with your own
4. Paste plugin scripts into your Carrd Body End embed block
5. Publish

## Customisation

All colours and type settings use Design Tokens CSS variables.
Edit the token block in Body Start to rebrand the entire site at once:

```css
:root {
  --color-primary:  #000000; /* your brand colour */
  --color-accent:   #7C5CBF; /* your accent */
  --font-heading:   'Your Font', sans-serif;
}
```
