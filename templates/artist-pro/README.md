# 2urn Artist Pro Template

**Status:** Demo  
**Demo:** Available via Carrd dashboard  
**Price:** $49  
**Markets:** Fine Artists, Visual Artists, Photographers, Illustrators

## Overview

A premium Carrd template built for visual artists who need archive-quality documentation
of their work — clean gallery, artist CV, statement, and contact — without Behance's
audience lock-in or Instagram's algorithmic dependency.

## Features

- Full-width image-first hero
- Filterable gallery (Gallery Filter Pro+ with category tags: Painting, Drawing, Digital, Installation, etc.)
- Artist statement section with large typographic treatment
- CV section — Education, Exhibitions, Press, Collections (Clone Engine repeatable)
- Shader FX or Gradient FX hero background option
- SVG FX decorative elements
- Water Glass Engine for overlay panels
- Mobile-first, fully responsive

## Sections

1. **Hero** — Full-bleed image or Shader FX background, name, medium
2. **Work** — Filterable gallery grid (Gallery Filter Pro+ categories)
3. **Statement** — Artist statement, large type
4. **CV** — Exhibitions, Education, Press, Collections (repeatable entries)
5. **About** — Bio, portrait, studio location
6. **Contact** — Inquiry form, representation info

## Plugins Included

- Gallery Filter Pro+ v1.7.3
- Clone Engine v2.0
- Design Tokens v1.4
- Shader FX v1.0
- SVG FX v2.0
- Water Glass Engine v1.0

## Installation

1. Purchase the template at [2urn.com](https://2urn.com)
2. Duplicate the Carrd template to your account
3. Replace placeholder content and images
4. Paste plugin scripts into Carrd Body End embed block
5. Set your gallery item `data-category` attributes
6. Publish
