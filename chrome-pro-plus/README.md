# 2urn · Carrd Chrome Pro+ (v1.1)

Sticky header with fade/shrink and automatic mobile slide-out navigation for Carrd sites.

## Features
- Sticky header with fade and shrink on scroll
- Content overlay support for transparent headers
- Automatic mobile menu with slide-out navigation

## Installation
1. Go to `chrome://extensions` → Enable Developer Mode
2. Click "Load unpacked" → select this folder
