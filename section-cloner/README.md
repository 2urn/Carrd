# 2urn · Section Cloner (v1.0)

Chrome extension that clones entire Carrd sections (scrollpoint anchor + all containers) with one click.

Solves a core Carrd limitation: the platform only clones individual containers, not full sections with their anchor scrollpoints.

## Features
- Floating editor panel in the Carrd editor
- Click-to-select sections with visual highlighting
- Clone above or below in one click
- Alt+C / Alt+Shift+C keyboard shortcuts
- Template save/paste via chrome.storage

## Installation
1. Go to `chrome://extensions` → Enable Developer Mode
2. Click "Load unpacked" → select this folder
